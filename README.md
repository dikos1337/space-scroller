# Space Shooter
Игра где ты управляешь космическим кораблем и уворачиваешься от метеоритов.

![alt text](https://i.imgur.com/FPQBCBT.png)

# Запуск на Windows
```sh
git clone https://github.com/dikos1337/space-scroller
cd space-scroller
python -v venv venv
venv\Scripts\activate 
pip install -r requirements.txt
python src/main.py
```
